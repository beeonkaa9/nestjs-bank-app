## Description

A bank app created with the [Nest](https://github.com/nestjs/nest) framework. Create accounts, as well as withdraw, add, and transfer money to other accounts!

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
